# Website Analysis

## Target Website

**Website:** https://books.toscrape.com/

Books to Scrape is a publicly available website designed for practicing web scraping. It provides structured HTML, making it suitable for building and testing data extraction pipelines.

## Data Collected

The scraper extracts the following information for each book:

- Book Name
- Price
- Rating
- Availability
- Product URL

## Observations

- The website contains 50 pages of books.
- Each page contains 20 books.
- Total books scraped: **1000**.
- The HTML structure is consistent across all pages, enabling reliable automated scraping.
