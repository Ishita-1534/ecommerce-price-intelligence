# HTML Parsing with BeautifulSoup

## Purpose

BeautifulSoup is used to parse the HTML content of the Books to Scrape website and extract structured book information.

## Information Extracted

- Book Name
- Price
- Rating
- Availability
- Product URL

## BeautifulSoup Methods Used

- `BeautifulSoup()`
- `select()`
- `select_one()`
- `text`
- `strip()`
