# Data Dictionary

| Column | Data Type | Description |
|---------|-----------|-------------|
| id | INT | Auto-increment primary key |
| product_name | VARCHAR(500) | Name of the book |
| price | DECIMAL(10,2) | Book price (£) |
| rating | VARCHAR(20) | Book rating (One to Five) |
| availability | VARCHAR(100) | Stock availability |
| product_url | TEXT | Relative URL of the book |
